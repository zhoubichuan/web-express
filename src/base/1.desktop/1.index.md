# Express 介绍

* Exprss 是一个简洁、灵活的 node.js Web 应用开发框架，是目前最流行的基于 Node.js 的 Web 开发框架，他提供一些列强大的功能，比如：
  * 模板解析
  * 静态文件服务
  * 中间件
  * 路由控制
* 还可以使用其他模板来帮助你创建各种 Web 和移动设备应用
